Gemini API 무료 발급 연동 가이드

  1. Google AI Studio (aistudio.google.com) 이동
  2. 구글 계정 로그인 후 왼쪽 상단 "Create API Key" 클릭
  3. 새 프로젝트 또는 기존 프로젝트용 API 키를 생성하고 복사합니다.
  4. 복사한 키를 위 입력란에 붙여넣고 "저장 및 부스터 활성화"를 눌러주세요.

※ 입력하신 API 키 정보는 서버의 메모리 상에서만 1회성 중계에 사용되며, 안전하게 귀하의 브라우저 로컬 저장소(localStorage)에만 암호화 및 보관됩니다. 또한, 2km 이내의 진짜 식당 정보만을 엄선하여 보여줍니다.
